# KKCardModCheck v1.0.0 发布!!


> 在最近的更新中添加了全新的界面,使用上更方便了!

**KKCardModCheck** 是一款用于检测 Koikatsu 系列游戏角色卡的 MOD 工具，旨在帮助玩家快速解析角色卡信息、检测缺失的 MOD，并生成相关的借物表。


[**下载地址**](https://github.com/GenesisAN/KKCardModCheck/releases)

---

## 功能列表

- [x] **角色卡解析**
  - 支持 KK 和 KKS 系列角色卡解析
  - 支持服装卡解析
- [x] **MOD 检测**
  - 单个角色卡的 MOD 缺失检测
  - 批量角色卡的 MOD 缺失检测
  - 角色卡使用 MOD 验证
- [x] **借物表生成**
  - 自动生成角色卡的借物表
- [x] **秒传信息检索**
  - 检索角色卡的秒传信息
- [x] **更新检查**
  - 检测工具的最新版本
- [ ] **秒传信息上传**（开发中）
- [ ] **本地游戏 MOD 信息读取与上传**（开发中）

## 使用说明

1. **将软件解药到任意文件夹内点击exe启动**：
2. **设置游戏所在目录**：
   - 如果是第一次启动软件,会跳出欢迎页面,需在此设置游戏目录
   - 游戏目录是指 Koikatsu 或 Koikatsu Sunshine 安装目录,例如 `D:\SteamLibrary\steamapps\common\Koikatsu` 或 `D:\SteamLibrary\steamapps\common\Koikatsu Sunshine`
3. **构建Mod信息**：
   - 点击“构建MOD信息”按钮，软件会自动扫描游戏目录下的 MOD 并生成 MOD 信息数据库。
   - 如果你添加了新的 MOD，也可以随时重新构筑 MOD 信息。
4. **开始使用**：
   - 至此你就可以使用软件的各项功能了，例如解析角色卡、检测 MOD 缺失等。
   - 如果你想重置软件,可以通过设置页面中的`重置`按钮来完成。也可以通过删除软件所在目录下的 `KKCardModCheck_Data` 文件夹来重置。
