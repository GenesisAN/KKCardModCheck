# KKCardModCheck v1.0.0 发布!!


> 在最近的更新中添加了全新的界面,使用上更方便了!

**KKCardModCheck** 是一款用于检测 Koikatsu 系列游戏角色卡的 MOD 工具，旨在帮助玩家快速解析角色卡信息、检测缺失的 MOD，并生成相关的借物表。


[**下载地址**](https://github.com/GenesisAN/KKCardModCheck/releases)

---

## 功能列表

- [x] **角色卡解析**
  - 支持 KK 和 KKS 系列角色卡解析
  - 支持服装卡解析
- [x] **MOD 检测**
  - 单个角色卡的 MOD 缺失检测
  - 批量角色卡的 MOD 缺失检测
  - 角色卡使用 MOD 验证
- [x] **借物表生成**
  - 自动生成角色卡的借物表
- [x] **秒传信息检索**
  - 检索角色卡的秒传信息
- [x] **更新检查**
  - 检测工具的最新版本
- [ ] **秒传信息上传**（开发中）
- [ ] **本地游戏 MOD 信息读取与上传**（开发中）

## 使用说明

1. **将软件放入游戏目录下启动**：
   - 第一次启动时会提示生成 `start.bat` 文件。
2. **双击 `start.bat` 文件启动软件**：
   - 该文件会设置 CMD 的字符串格式为 UTF-8，避免界面渲染问题。
3. **操作说明**：
   - 按下对应按钮启动功能，或使用小键盘上下键切换功能选项，按回车确认。
   - 在多个输入框之间切换时，按 `TAB` 键。
   - 弹窗界面选择按钮时，按左右键切换选项。
